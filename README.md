# Meeting Intelligence Hub

## The Problem
Organizations hold dozens of meetings weekly, producing long transcripts. Nobody has time to read 20+ pages to find key decisions or action items, causing confusion and wasted time.

## The Solution
This platform ingests multiple meeting transcripts, automatically extracts decisions and action items (who, what, by when), provides a chatbot to ask natural language questions across meetings, and visualizes speaker sentiment and tone over time.

## Tech Stack
- Programming languages: Python, JavaScript
- Frameworks: FastAPI (backend), React (frontend)
- Databases: PostgreSQL
- APIs: OpenAI API (for extraction & chatbot)
